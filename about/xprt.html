'Selling' the Upsdell site
It's all about ease

Web surfers, regardless of age, want their life to be easy. It is therefore our main goal to make the website as easy to use as possible so that everyone benefits.

It has what you want

The Upsdell site has more information available than any other house sites. This means the availability of:

  • Events
    • Rehearsal times
    • Progress tracking and practice toolkits
    • Results
  • Captains
    • Searching by event
    • Looking up by name
  • Photo, Video and cultural impact

Most importantly, this site fulfills every part of the specification as posted on the KGV main site before school (2007) started.

Surf without getting lost
  • A back hyperlink is available on every page so that the users know where they are and where they went, and that they won't be scared while surfing deeper into the site for more interesting information.
  • The website structure is amazingly simple and is URL-friendly. For example, http://www.kgv.edu.hk/upsdell/captains gets you exactly what you would expect.
  • Visited hyperlinks are, as recommended, grayed/greyed out.
  • Dynamically adjustable font sizes are there for the older audience.
    • As for the default font size, since the major target is the student body in the school itself, it has been set to a smaller size (10 pixels) than mature content sites. This is also the same font size as established sites, such as MSN, Yahoo and Slashdot.
    • The site has been made like other web pages as to reduce the learning time for new visitors.
No more than 3 clicks
  • The Upsdell website contents are, as recommended, nothing more than 3 clicks away.
  • Certain pages with heavy content, such as the captains page or the events page, come with either a searching script or an interactive display that will help the user understand what they are doing.
  • Older content are flushed out of the home page and is put elsewhere so the first impression is never cluttered.
  • We give people credits for helping the Upsdell web site, and make it clear to people what they can copy by using a copyright notice.
It always shows
  • Having been rigorously tested with online agents, this site is written in XHTML in such a way that it shows just as it is on every browser.
  • Anyone with JavaScript disabled can still surf the site.
  • This site is CSS 2.1 compliant and adds one line of IE-specific code for handling transparency.
